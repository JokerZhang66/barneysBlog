1.Vscode集成git-Bash终端

使用ctrl + shift + p打开设置,再点开右上角的settings.json,填入下列配置

1
2
3
4
5

"terminal.integrated.profiles.windows": {
    "Git-Bash": {
      "path": "F:\\Git\\bin\\bash.exe"
    }
 }

设置默认终端可以使用下列语句：

1

"terminal.integrated.defaultProfile.windows": "Git-Bash"
