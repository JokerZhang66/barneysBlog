### 本地安装vercel

输入下面的命令:
```
npm -v # 检查是否安装npm
npm i -g vercel # 全局安装vercel
vercel -v # 查看vercel版本，检查是否安装成功
```

### 部署项目
在自己的本地项目的根目录输入`vercel`命令进行部署，按照命令提示输入
